# kranio_interview

This is an interview project created for Kranio

## What's OK

MicroService1 returns object with "number" and "type" (odd or even).
Example: http://localhost:3001/sendNumber

- Body: {"number": 2}
- Response: {"number": 2, "type": "even"}
- Body: {"number": 3}
- Response: {"number": 3, "type": "odd"}
